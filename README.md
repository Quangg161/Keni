## Mô tả Dự Án

KENI là một website thương mại điện tử đơn giản cho phép người dùng:
- Duyệt và tìm kiếm sản phẩm giày
- Xem chi tiết sản phẩm, chọn size và số lượng
- Thêm vào giỏ hàng
